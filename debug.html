Login Debugging

If you're having issues with login, try these steps:

  1. Clear your browser cache and cookies
  2. Try using the direct login URL: http://localhost:8084/login
  3. Use the admin demo account: admin@example.com / password123

Authentication Flow

The authentication flow should work as follows:

1. User enters credentials on login page
2. AuthContext.login() is called
3. Supabase authenticates the user
4. AuthContext fetches the user profile
5. User is redirected to dashboard

Manual Authentication Test

You can test the Supabase authentication directly using the console:

// Open browser console (F12) and paste this code:
const { data, error } = await fetch('http://localhost:8084/api/login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ 
    email: 'admin@example.com', 
    password: 'password123' 
  })
}).then(res => res.json());

console.log({ data, error });
